## 基本说明

这是一个可以通过Alfred 的关键词快速获取[SubHD.com](http://subhd.com/) 首页热门影视的Alfred Workflow。使用Python 语言编写，支持最新的Alfred 3。

使用本Workflow 前请确保你的Alfred APP 的Powerpack 功能已经解锁。
